## 💬 무엇을 새롭게 알았는지 
### HashMap
- HashMap은 Key와 Value값을 가진다.
- HashMap을 사용하려면 객체를 import해야 한다.
  ```java
  import java.util.HashMap;
  ```
- HashMap 생성
  ```java
  HashMap<KEY타입, VALUE타입> 객체명 = new HashMap<>();
  ```
- HashMap 값 추가 : `.put()`메서드
  ```java
  해시맵.put(키, 값);
  ```
- HashMap 값 가져오기
  ```java
  해시맵.get(키);
  ```
  - 원래 없던 키 값 가져오기
    ```java
    해시맵.getOrDefault(키, 디폴트값);
    ```
- HashMap 크기
  ```java
  해시맵.size();
  ```

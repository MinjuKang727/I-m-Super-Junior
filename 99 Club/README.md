### 🏆 출석률 랭킹전
매주 출석률이 가장 높은 반을 선정해 반 전원에게 보상을 드려요.   
같은 그룹 내에서 선의의 경쟁을 해 봐요!

`자바/비기너`, `자바/미들러`, `자바/챌린저`, `파이썬/비기너`, `파이썬/미들러`, `파이썬/챌린저` 각 그룹별로 우수 출석반을 선정.  
*(=매주 최소 6개 반 이상)*

같은 그룹 내에서 가장 높은 출석률이 2개반 이상이라면 해당반 모두 우수 출석반!

*출석률: ( 실제 출석일수 / (팀원 수X의무 출석 일수) ) X 100 (%)*
